# IPL Data Analysis Dashboard

## Project Overview

This project analyzes historical IPL matches (2008-2025) using Power BI.

The dashboard helps users analyze

• Team Performance
• Player Statistics
• Match Results
• Toss Impact
• Venue Analysis
• Season-wise Performance
• Interactive Filters
• Business Insights

---

## Tools Used

- Power BI
- DAX
- Power Query
- Excel
- GitHub
- Kaggle

## Team Analysis

<img width="923" height="518" alt="Screenshot 2026-07-09 105834" src="https://github.com/user-attachments/assets/9f66307a-c235-4b66-a04d-475ba4c46a46" />

This page focuses on team performance across IPL seasons. Users can compare teams based on matches played, wins, runs scored, and other performance metrics. Interactive filters allow quick comparison between teams and seasons.

## Player Analysis

<img width="928" height="518" alt="Screenshot 2026-07-09 110023" src="https://github.com/user-attachments/assets/3e01e8f4-1e67-4ccd-aa5a-ce91f4287e0e" />

The Player Analysis dashboard provides insights into individual player performances. It highlights top run scorers, leading wicket takers, strike rates, sixes, fours, types of bowler wickets and other key performance indicators to evaluate player contributions across different seasons.

## Venue Analysis

<img width="927" height="521" alt="Screenshot 2026-07-09 110137" src="https://github.com/user-attachments/assets/70d031fd-0e12-4241-bff7-b7be0b442c7b" />

The Venue Analysis dashboard examines how different IPL venues have influenced match outcomes and team performances. It provides insights into venue-specific trends such as the number of matches hosted, highest score, highest wicket taking venues, highest first and second inning scores, average scores and boundaries. Interactive filters allow users to compare venues across different seasons and identify stadiums that favor batting or bowling conditions.
